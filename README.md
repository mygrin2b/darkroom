# DARKROOM

一款线框风格的黑暗环境第一人称射击游戏。在几乎全黑的空间里，你只能靠手电、枪口闪光和弹道曳光看清周围——但开灯的同时也会暴露自己。

## 玩法

- **战役**：潜入黑暗的房间，清除敌人，找到出口。
- **对战战区**：1v1 在线对战，进入战区自动与另一名玩家配对。想和好友对战，双方填写相同的房间号即可。

## 操作

| 操作 | 按键 |
|------|------|
| 移动 | W A S D |
| 慢走（静音） | 按住 Shift |
| 开关手电 | T |
| 开火 | 鼠标左键 |
| 精确瞄准 | 鼠标右键 |
| 换弹 | R |
| 互动 | F |
| 地图 | M |
| 暂停（仅战役） | P |

## 在线对战说明

对战通过 WebRTC 建立玩家间的点对点直连，游戏数据不经过任何服务器。自动配对仅借助公共信令中继牵线，连接建立后即断开。若双方处于无外网环境，可使用界面中的「手动交换连接码」方式连接。

## 技术

纯静态单文件，基于 Three.js 渲染。无需构建，直接用浏览器打开 `index.html` 即可运行。
